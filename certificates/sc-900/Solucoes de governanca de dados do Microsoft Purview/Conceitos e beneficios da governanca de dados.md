# conceitos e benefícios da governança de dados

Historicamente, a governança de dados tem sido um mecanismo de defesa, uma maneira de garantir que os seus dados estejam seguros e em conformidade. Mas uma boa governança de dados também torna os seus dados mais visíveis para os seus usuários e oferece muitas oportunidades para reunir seus negócios com os dados que os alimentam.

## Conceito de governança de dados

- **Governança de dados federada:** Centraliza padrões de segurança, qualidade e políticas, enquanto distribui a responsabilidade pelos dados entre as áreas, reduzindo gargalos e incentivando o autoatendimento.
- **Acesso a dados:** Garante acesso rápido e adequado aos dados corretos, equilibrando segurança e inovação.
- **Curadoria de dados:** Organiza, documenta e publica os dados para torná-los acessíveis, reutilizáveis e seguros.
- **Descoberta de dados:** Permite que os usuários encontrem facilmente os dados necessários para atividades de negócio e inovação.
- **Integridade dos dados:** Assegura a qualidade, atualização e proteção dos dados ao longo de todo o seu ciclo de vida.
- **Compreensão de dados:** Fornece descrições e indicadores de qualidade para que os usuários entendam os dados e saibam como utilizá-los corretamente.

## Funções e responsabilidades de dados

- Os consumidores de dados encontram e usam reapidamente conjustos de dados confiáveis relevantes por meio de um fluxo de trabalho de solicitação de acesso simplificado.
- Os proprietários de dados registram ativos de dados para uso, gerenciam classificações e acesso e garantem padrões de alto qualidade
- Os administradores de dados garantem a qualidade dos dados, a descorberta contínua de dados, a consistência do glossário e a linhagem.
- O escritório de dados central estabelece e garante políticas de governança, metadados arivos, conformidade e insights sobre a integridade geral da governança.

# Benefícios da governança de dados

Uma boa governança de dados ajuda a revelar o valor comercial de seus dados e simplifica o gerenciamento de dados à medida que o seu patrimônio de dados cresce. Ela oferece benefícios importantes para as diferentes funções de dados:

- Para consumidores de dados em toda a organização:
  - Descoberta de dados – ajuda você a encontrar facilmente os dados necessários.
  - Acesso seguro – facilita o acesso seguro aos seus dados.
  - Compreensão de dados – fornece o que você precisa saber sobre os dados.
- Para proprietários de dados e administradores:
  - Curadoria e gerenciamento de dados – ajuda você a fornecer dados de alta qualidade que são fáceis de entender e acessar com segurança para aplicativos de toda a organização.
  - Uso responsável de dados – ajuda você a garantir que os seus dados sejam usados pelos usuários pretendidos para os fins pretendidos.
  - Análise de impacto – entenda as ações nos dados que podem afetar os seus dados.
- Para oficiais de dados e stakeholders de CxO:
- Criação de valor de dados – maximize a criação de valor de seus dados ao mesmo tempo em que reduz os gastos com operações.
- Padronização de propriedade de dados – crie controles comuns em seu conjunto de dados com responsabilidade federada para que os seus dados sejam íntegros e seguros.
