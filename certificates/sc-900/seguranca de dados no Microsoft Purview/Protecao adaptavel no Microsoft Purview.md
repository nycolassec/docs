# Proteção adaptável no Microsoft Purview

A proteção adaptável no Microsoft Purview usa aprendizado de máquina (ML) para identificar os riscos mais críticos e aplicar controles de proteção de forma proativa e dinâmica a partir de:

- Prevenção contra perda de dados
- Gerenciamento do Cicle de Vida dos Dados do Microsoft Purview
- Acessso condicional do Microsoft Entra

A integração com prevenção contra perda de dados, gerenciamento do ciclo de vida dos dados e Acesso Condicional pode ajudar as organizações a automatizar suas respostas aos riscos internos e reduzir o tempo necessário para identificar e corrigir ameaças potenciais.

A proteção adaptável ajuda a mitigar riscos potenciais usando:

- Detecção com reconhecimento de contexto. Ajuda a identificar os riscos mais críticos com análise baseada em ML tanto de conteúdo quanto de atividades do usuário.
- Controles dinâmicos. Ajuda a impor controles eficazes em usuários de alto risco, enquanto os demais mantêm a produtividade.
- Mitigação automatizada. Ajuda a minimizar o impacto de potenciais incidentes de segurança de dados e a reduzir a sobrecarga administrativa.

A proteção adaptável atribui dinamicamente políticas apropriadas de prevenção contra perda de dados, gerenciamento do ciclo de vida dos dados e Acesso Condicional aos usuários com base nos níveis de risco interno (elevado, moderado ou menor), definidos e analisados pelos modelos de aprendizado de máquina no Gerenciamento de Risco Interno.

## Proteção adaptável na Prevenção contra Perda de Dados

A Proteção Adaptável no Microsoft Purview integra o Gerenciamento de Risco Interno do Microsoft Purview com a Prevenção contra Perda de Dados (DLP) do Microsoft Purview. Quando o Gerenciamento de Risco Interno identifica um usuário que está se comportando de forma arriscada, ele é dinamicamente atribuído a um nível de risco interno.

Em seguida, a proteção adaptável pode criar automaticamente uma política DLP para ajudar a proteger a organização contra o comportamento suspeito associado a esse nível de risco interno.
